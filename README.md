Created a tableau dashboard with 3 different files orders , Managers , Returns and combined them with join in tableau

Dataset Description
# Orders.csv
Contains order-level sales data:

Order ID ,
Customer info , 
Product info , 
Sales, Profit, Quantity
Region, Segment, Category,

# Returns.csv
Indicates whether an Order ID was returned.

# Managers.csv
Contains Region → Manager mapping.

# Visualizations Included
1. Sales by Region (Bar Chart)
Compares regional sales and displays responsible managers.

2. Monthly Sales Trend (Line Chart)
Shows seasonality and sales growth.

3. Category vs Profit (Bar Chart)
Highlights profitable and unprofitable product categories.

4. Returns by Region (Stacked Bar)
Compares how return behavior differs across regions.

5. Sub‑Category Profitability (Scatter Plot)
Shows relationship between Sales and Profit for every product sub-category.
